# info

The info API provides functions to list available rules and dialects.

## Functions

### list_dialects

```python
list_dialects() → list[DialectTuple]
```

Return a list of available dialect info.

**Returns:**

`list[DialectTuple]` — See return type above

### list_rules

```python
list_rules() → list[common.RuleTuple]
```

Return a list of available rule tuples.

**Returns:**

`list[common.RuleTuple]` — See return type above
